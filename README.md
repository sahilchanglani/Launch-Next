# Launch Next
<br/>
Launch Next is an online platform to register and promote startups. Launch Next publishes the newest, trending tech startups and projects.

## Features

- Login/Sign Up
- Register Startup
- Search by tags
- Search by startup name
- Edit, delete and like startups.
